# お題追加の手順

お題データは `src/lib/prompts.ts` の `prompts` 配列にある。

## 形式

```ts
{ id: '60', text: '〜を話してみよう', category: '既存カテゴリ名', starter: 'そのまま読める話し出し例。' },
```

- `id`: 既存の最大値 +1 の連番(文字列)。既存のidと重複しないこと
- `text`: 「〜を話してみよう」の形に揃える。命令形にしない
- `category`: 既存カテゴリ(今日の予定 / 今の気分 / 好きなもの / 出来事 / 自己紹介 / 説明練習 など)に入れる。新カテゴリを作る場合はカテゴリ内で最低5件は用意する
- `starter`: 必ず付ける。ユーザーが「そのまま声に出して読める」文。〇〇を埋めるだけで話し出せる形にする

## 文言のルール(最重要)

- 評価語(上手に、正しく、しっかり)を使わない
- 答えにくいお題(トラウマ、人間関係の失敗など)を出さない
- 一つのことだけ聞く。「AとBを話そう」にしない
- 広報部(writer)のトーンに従う: やさしく、責めない

## 追加後の確認

1. `npm run typecheck` と `npm run lint` を実行
2. README.md にお題件数の記載(「59件のお題」など)があれば数を更新する
